1) The rule

A whole number is divisible by 2 if and only if its last digit is even (0, 2, 4, 6, 8).

“Divisible” means it goes in exactly with no remainder.

2) Why the last digit decides it (simple proof)

Write a number in base-10 (decimal) as digits a₄a₃a₂a₁a₀. Its value is

N = a₄·10⁴ + a₃·10³ + a₂·10² + a₁·10 + a₀.

  1. 10 is even. Since 2 | 10, every power 10², 10³, … is also even (divisible by 2).
  2. All except the last term have a factor 10. So a₄·10⁴ + a₃·10³ + a₂·10² + a₁·10 is a multiple of 2.
  3. Only the last digit can change even/odd. Therefore N is divisible by 2 iff a₀ is even.
One-line “mod” version (optional)

Because 10 ≡ 0 (mod 2), we have N ≡ a₀ (mod 2). So the whole number has the same remainder as its last digit when dividing by 2.
